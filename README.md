# HotS Draft Assistant

Десктопный помощник для фазы драфта в Heroes of the Storm. Вы вручную вводите
игроков, выбранных героев, баны и карту, а программа ведёт локальную базу
статистики и подсказывает, кого банить и пикать.

Программа определяет «однокнопочных» игроков (OTP), строит пул героев каждого
игрока, оценивает силу команд и выдаёт ранжированные рекомендации.

**Терминология:** левая колонка — ваша команда, правая колонка — команда
противника.

## Возможности

* Ручной ввод игроков, героев, банов и карты прямо в окне.
* Интегральная оценка силы игрока (рейтинг и буквенный грейд S–D).
* Определение OTP-игроков и построение пула героев с приоритетами.
* Матрицы синергий и контрпиков из истории матчей.
* Рекомендации по банам (приоритет — OTP-герои противника) и по пикам.
* Прогноз вероятности победы и полупрозрачный оверлей поверх игры.

## Требования

* Python 3.9.13 (поддерживается 3.9+)
* Один Python-пакет: **PyQt5** (окно приложения и оверлей)

Никаких системных программ, OCR или дополнительных библиотек не нужно.

## Установка

```bash
git clone <repository-url>
cd hots_draft_assistant
python -m pip install -r requirements.txt
```

Если команда `pip` не распознаётся — используйте `python -m pip`.
Если не распознаётся `python` — попробуйте `py` (Python Launcher на Windows).

## Запуск

Открыть окно приложения:

```bash
python main.py
```

Быстрая текстовая проверка без окна:

```bash
python main.py --console
```

## Как пользоваться

После `python main.py` открывается окно с двумя колонками: слева ваша команда,
справа команда противника.

1. **Введите игроков и героев.** Кликните по любому слоту — откроется окно
   ввода: battletag игрока и выпадающий список героев с поиском. Кнопка
   «Очистить» сбрасывает слот.
2. **Отметьте себя.** Нажмите «Это я» на своём слоте — рекомендации по пикам
   считаются именно для этого игрока. Активный слот подсвечивается золотом.
3. **Заполните баны.** Над каждой колонкой есть три ячейки банов — кликните и
   выберите героя.
4. **Выберите карту** в выпадающем списке сверху.
5. **Смотрите рекомендации.** Они пересчитываются автоматически через ~0.4 с
   после любого изменения. Можно нажать «⚡ Рассчитать» вручную. Кнопка
   «↺ Сброс» очищает всё.

В центре показывается средний MMR команд и полоса вероятности победы. Снизу —
карточки рекомендаций: баны (★ = OTP-герой противника) и пики (с оценкой и
причиной: OTP / пул / контрпик / синергия / мета).

### Демонстрационные сценарии

В шапке окна две кнопки с примерами (данные подставляются автоматически):

* **🎬 Демо 1** — стадия сбора составов: 2 игрока в своей команде, 3 игрока у
  противника и по 3 бана с каждой стороны.
* **🎬 Демо 2** — середина драфта: по 2 бана с каждой стороны, 1 пик у
  противника и 1 пик в своей команде.

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `main.py` | Точка входа: запускает окно или текстовую проверку (`--console`). |
| `app.py` | Окно приложения (PyQt5): колонки команд, баны, рекомендации, журнал. |
| `engine.py` | Ядро: сессии драфта, аналитика, рекомендации, демо-сценарии. |
| `config.py` | Настройки: веса формул, пороги OTP, параметры рейтинга. |
| `db.py` | База данных SQLite: игроки, герои, пул, сессии, рекомендации. |
| `schema.sql` | Схема таблиц базы данных. |
| `data_sources.py` | Справочник героев и карт, импорт из парсера реплеев. |
| `analytics.py` | Рейтинг игрока, определение OTP, пул героев, анализ ролей. |
| `recommendations.py` | Алгоритм рекомендаций по банам и пикам. |
| `overlay.py` | Оверлей поверх игры и окно настроек. |
| `requirements.txt` | Единственная зависимость — PyQt5. |

## Как наполняется база данных

База стартует пустой (только справочник героев и карт). Статистика игроков
добавляется через парсер реплеев на базе
[heroprotocol](https://github.com/Blizzard/heroprotocol): он разбирает файлы
`.StormReplay` и передаёт матчи в `data_sources.import_from_replay_parser`.
Пока статистики нет, игроки получают нейтральную оценку, а рекомендации
строятся по справочным данным.

## Часто задаваемые вопросы

**`'python' is not recognized`** — Python не добавлен в PATH. Переустановите
Python с галочкой «Add Python to PATH» и перезагрузите компьютер, либо
используйте `py` вместо `python`.

**`'pip' is not recognized`** — используйте `python -m pip` вместо `pip`.

**`No module named PyQt5`** — выполните `python -m pip install PyQt5`.

**Рекомендации не появляются** — заполните хотя бы несколько слотов и нажмите
«Это я» на своём слоте, либо нажмите кнопку «🎬 Демо 1».
